# WAD files

Place a legal WAD file here as:

```text
public/wads/doom1.wad
```

Use the shareware `doom1.wad` or a WAD you are legally allowed to use. Do not
upload or commit the commercial `DOOM.WAD` unless you have the legal right to do
so, and do not publish it publicly.

The browser app will try `/wads/doom1.wad` first. If the file is missing, the
WebAssembly DOOM module falls back to its built-in shareware WAD.
